Sydney Trains Data Visualisation System

close
chevron_left chevron_right
Features:
  • Identifies current trends of the entire train's network
  • Compares scheduled and actual performance to trace the starting point of delay
  • Displays the patterns of delay for different times in each station
  • Specific queries like "Most Frequently Late Trips" and "Number of Trains that Skipped Stops" can be searched by interpreting historical trains data
  • Shows the whole real-time train's network in an interactive map by Leaflet.js and OpenStreetMap ➔ bottleneck can be spotted instantly if formed
  • Downloads datasets as CSV format
  • Utilizes HTML, CSS, jQuery, Node.js, MySQL, Chart.js, Leaflet.js, etc.
